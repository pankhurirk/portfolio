## Design
Edit `assets/styles.css`.

The main colour variables are at the top:

```css
--paper: #f4f0e8;
--ink: #171713;
--muted: #625f57;
--accent: #9f3d2e;
```

The current version intentionally uses no external fonts or libraries, so it remains fast and simple to host.
